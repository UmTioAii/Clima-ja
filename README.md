🌤️ Clima Já
Aplicação web moderna e responsiva para consultar a previsão do tempo em tempo real de qualquer cidade do mundo. Desenvolvida com Vite e React, oferece uma interface limpa e informações meteorológicas atualizadas.

✨ Funcionalidades
Busca por Cidade: Permite pesquisar e visualizar o clima atual de qualquer cidade do mundo.

Informações Completas: Exibe dados como temperatura, umidade, velocidade do vento, descrição do clima (ex: "céu limpo") e sensação térmica.

Interface Responsiva: Design adaptado para funcionar perfeitamente em celulares, tablets e desktops.

Rápido e Leve: Construído com Vite para um desempenho otimizado e carregamento rápido.

🚀 Tecnologias Utilizadas
Vite: Ferramenta de build que proporciona um ambiente de desenvolvimento mais rápido.

React: Biblioteca JavaScript para construção de interfaces de usuário.

CSS Modules (ou Styled Components): Para estilização componentizada e isolada. (Ajuste se estiver usando outra abordagem)

API de Clima: Integração com uma API externa de previsão do tempo (ex: OpenWeatherMap, WeatherAPI). (Lembre-se de configurar sua chave)
